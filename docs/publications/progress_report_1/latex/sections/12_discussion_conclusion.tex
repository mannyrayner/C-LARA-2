\paragraph{Workflow limitation to discuss.}
The current Codex-based publication workflow has a practical limitation that
should be recorded in the final discussion: this environment can inspect
images supplied in conversation, but its pull-request path does not reliably
accept binary image assets. The report has therefore used textual figure
placeholders while real screenshots are prepared separately for Overleaf.
This limitation does not change the platform claims, but it is evidence that
end-to-end autonomous repository maintenance and report production still
encounter tooling assumptions that are not well suited to this workflow.
